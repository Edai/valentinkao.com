Stripe and Visa shipped the Machine Payments Protocol on March 18. An open standard for AI agents to pay autonomously. 100+ services in the directory at launch.

- Browserbase: agents pay per headless browser session.

- PostalForm: agents pay to print and send physical mail.

- Dune Analytics: agents pay per query.

More and more infra for "agentic economy" is getting live while others like Walmart are using it for discovery (check my last Tuesday post!)

This changes one assumption most payment systems make: a human is in the loop.
