# valentinkao.com

**Description:** NextJS portfolio

## Getting Started

### Installation

1. Clone the repository:
```
git clone https://github.com/Edai/valentinkao.com
```

2. Navigate to the project directory:
```
cd valentinkao.com
```

3. Install the dependencies:
```
npm install
```

4. Create the env file:
```
NOTION_SECRET=
NOTION_DB_ID=
NEXT_PUBLIC_GOOGLE_ANALYTICS=
```

### Running the Project

- **Development:** 
```
next dev
```

- **Build:** 
```
next build
```

- **Start:** 
```
next start
```
